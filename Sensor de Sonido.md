# Sensor de Sonido

## Materiales
          -Placa Arduino Mega 2560
          -Conector de placa Arduino
          -Protoboard
          -Resistencia 220ohm
          -LDiodo Led
          -Jumpers
          -Sensor ultrasonico HC-SR04
          
          
## Funcionamiento
